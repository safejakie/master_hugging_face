## Installation

1. Clone le projet
   git clone https://github.com/safejakie/master_agent_IA.git

2. Copie le fichier .env.example et renomme-le .env
   cp .env.example .env

3. Remplis ton .env avec tes vraies clés :
   - Clé Groq → console.groq.com (gratuit)
   - Email Gmail + mot de passe application

4. Installe les dépendances Python
   pip install -r requirements.txt

5. Lance le serveur
   python server.py

6. Ouvre index.html dans ton navigateur
